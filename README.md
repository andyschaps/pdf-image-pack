# PDF Slide
Create conclude some images PDF slide.
PDF generated by [PDFKit](http://pdfkit.org/)

# Usage

```js
var PDFSlide = require("pdf-slide")

var imgs = [
  "./fixture/basic/a.png",
  "./fixture/basic/b.png",
]
var output = "./tmp/out.pdf"
var slide = new PDFSlide()
slide.output(imgs, output)

```
